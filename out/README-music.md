# Фоновая музыка

Чтобы включить музыку в игре, положите аудиофайл в корень `public/` с именем **`bg-music.mp3`**.

- Путь: `public/bg-music.mp3` → в приложении доступен как `/bg-music.mp3`
- Формат: MP3 (или другой поддерживаемый браузером)
- Рекомендуется: спокойный зацикленный трек без резкого конца (loop)

Кнопка вкл/выкл музыки отображается в правом нижнем углу экрана. Выбор пользователя сохраняется в `localStorage`.
